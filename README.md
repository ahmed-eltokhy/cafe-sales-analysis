# ☕ Cafe Sales Analysis Dashboard

## 📌 Project Overview

This project analyzes cafe sales data using Python and Tableau to transform raw data into meaningful business insights.

## 🔍 Project Workflow

* Data Cleaning & Preprocessing using Pandas
* Handling Missing Values and Unknown Data
* Exploratory Data Analysis (EDA)
* Building an Interactive Tableau Dashboard

## 🛠️ Tools Used

* Python
* Pandas
* Tableau

## 📊 Dashboard KPIs

* Total Revenue
* Total Quantity Sold
* Total Orders

## 📈 Key Insights

* Salad generated the highest revenue.
* Coffee was among the top-selling items by quantity.
* Sales fluctuated across different months.

* Raw Dataset
* Cleaned Dataset
